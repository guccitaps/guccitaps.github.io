Nos Inspirations

Pour créer Block'N'Roll, nous nous sommes inspirés de :

  • Minecraft bien entendu au niveau de l'esthétique du jeu. Ce jeu a lancé cette mode des jeux "cubiques".
  • Edge qui est le jeu qui se rapproche le plus de Block'N'Roll dans les déplacements du cube et dans le gameplay en général.

Logiciels Utilises

Pour Créer Block'N'Roll, nous avons utilisés :

  • Unity pour le développement du jeu.
  • Microsoft Visual Studio pour coder les scripts.
  • Gimp pour le logo du jeu.
